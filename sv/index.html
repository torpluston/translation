SNOWFLAKE

Snowflake är ett system för att besegra internet censur. Människor som är censurerade kan använda Snowflake för att komma åt internet. Deras anslutning går genom Snowflake-proxyer, som drivs av volontärer. För mer detaljerad information om hur Snowflake fungerar se vår dokumentation wiki.

Webbläsare

Om din internetanslutning censureras, bör du hämta Tor Browser.

Tillägg

Om din internetanslutning not censureras, borde du överväga att installera tillägget Snowflake för att hjälpa användare att censurera nätverk. Det finns ingen anledning att oroa sig för vilka webbplatser människor har tillgång till via din proxy. Deras synliga IP-adress kommer att matcha deras Tor-slutnod, inte din.

RAPPORTERA FEL

Om du stöter på problem med Snowflake som klient eller proxy, vänligen överväg att lämna in en felrapport. För att göra det måste du,

  1. Antingen skapa ett konto eller logga in med det delade kontot cypherpunks med lösenordet writecode.
  2. Skapa ett ärende med hjälp av vår felhanterare.

Försök att beskriva ditt ärende så bra som möjligt och om möjliga inkluderar loggmeddelanden som hjälper oss att reproducera felet. Överväg att lägga till sökord snowflake-webextension eller snowflake-client för att låta oss veta hur och vilken del av systemet Snowflake har problem.
